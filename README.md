# E-Commerce Orders - EDA Project

## Project Overview
Exploratory Data Analysis on a self-generated 
E-Commerce Orders dataset

## Dataset
- Rows: 1150
- Columns: 28
- Countries: 6 (India, USA, UK, 
  Australia, Canada, UAE)
- Source: Self generated using Python

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Analysis Performed
- Data Cleaning
- Univariate Analysis
- Bivariate Analysis
- Time Series Analysis
- Correlation Analysis
- Customer Loyalty Analysis
- Product Category Analysis

## Key Insights
- India has highest orders (27%)
- USA has highest avg order value
- Toys is top selling category
- Platinum tier generates most revenue

## Files
- data_generator.py → Dataset creation
- EDA_Ecommerce.ipynb → Python EDA
- Ecommerce_orders.csv → Raw dataset
- Ecommerce_orders_cleaned.csv → Cleaned data
- Ecommerce_Dashboard.pbix → Power BI dashboard
